# S3/OBS Connect

S3/OBS Connect is a Chrome extension for managing user-configured S3-compatible OBS buckets from a dedicated browser tab. It is intended for browsing bucket contents with **pagination support**, navigating prefixes, uploading files, previewing supported objects, downloading files, and removing objects without using a separate desktop client.

- Connect to an S3/OBS-compatible bucket using AK, SK, Access Path, and Server Address.
- Use **virtual-hosted-style** S3 access for bucket requests.
- Save connections in local Chrome extension storage.
- Open the app in a separate Chrome tab.
- Reuse an already open app tab when clicking the extension icon.
- Work with multiple connections in tabs inside the app.
- Switch between saved connections.
- Drag connection tabs to change their order.
- Close a connection with confirmation.
- View the current bucket path through path navigation.
- Navigate folder levels through path navigation.
- Load the list of folders and files in the current prefix.
- Search objects by prefix inside the current folder.
- Paginate object lists.
- Sort folders and files by name/type.
- Recursively delete a folder with its objects.
- Upload files with drag and drop.
- Show file upload progress.
- Preview text files/images/video/audio.
- Download a file from the preview window.
- Show and hide the local activity log.
- Light/dark theme.
