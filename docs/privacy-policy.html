Privacy Policy

Effective Date: May 20, 2026

This Privacy Policy describes how the S3/OBS Connect browser extension handles information.

1. Data Collection

The extension does not collect, sell, transmit, or share personal information with any project-owned server. There is no account system, backend service, analytics service, advertising service, payment processing, or AI/LLM API integration.

2. Local Data Usage and Storage

The extension stores user-entered connection details and local app state in the browser's isolated extension storage using chrome.storage.local. This can include access key IDs, secret access keys, OBS/S3 access paths, bucket names, prefixes, server addresses, inferred regions, connection tab order, current navigation state, local activity logs, and theme preference.

This local data is used only to reopen saved connections, connect to user-configured S3-compatible endpoints, browse bucket contents, upload files, create folders, preview supported files, download objects, delete objects, keep local activity history, and preserve the user's interface settings.

3. Network Requests

The extension sends storage API requests from the user's browser only to S3-compatible endpoints configured by the user. The endpoint address is provided by the user in the extension.

4. Downloads

When the user downloads an object, the extension creates a local browser download for the requested file. Downloaded files are saved by the browser according to the user's browser settings.

5. Remote Code

The extension does not load or execute remote code. All JavaScript, user interface code, and bundled libraries are included in the extension package.

6. Data Sharing

We do not sell, transfer, or share user data. Connection details and app state remain in local browser extension storage unless the user clears them, removes the extension, or exports data through normal browser or device mechanisms.

7. Security Notice

Users should enter only credentials they are authorized to use. Credentials are stored locally by Chrome extension storage on the user's device. Users can remove stored credentials by clearing extension data or uninstalling the extension.

8. Changes to This Policy

We may update this Privacy Policy by posting a new version on this page.
